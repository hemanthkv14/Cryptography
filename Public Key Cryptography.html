Public Key Cryptosystem with Authentication













Plain Text : Data which is in readable format and to be encrypted, Denoted by X

Cipher Text : Data which is in unreadable format and to be decrypted, Denoted by Y = E ( K , X )

Encryption Algorithm : The algorithm used to Encrypt data, take in the Plain Text and Key as input produce Cipher Text as output

Decryption Algorithm : The algorithm used to Decrypt data, take in the Cipher Text and Key as input produce Plain Text as output

PUX : Public key of person X.

PRX : Private key of person X.
